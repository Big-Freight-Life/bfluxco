## Gradients

### Brand Gradient
```css
/* Teal to Cyan - primary brand gradient */
--gradient-brand: linear-gradient(135deg, #14B8A6 0%, #06B6D4 100%);

/* Dark mode variant */
--gradient-brand-dark: linear-gradient(135deg, #2DD4BF 0%, #22D3EE 100%);
```

### Accent Gradient
```css
/* Purple to Coral - creative/warm gradient */
--gradient-accent: linear-gradient(135deg, #A855F7 0%, #F87171 100%);
```

### Mesh Gradient (Hero backgrounds)
```css
/* Complex mesh for hero sections */
--gradient-mesh: radial-gradient(at 20% 80%, rgba(20, 184, 166, 0.15) 0%, transparent 50%),
                 radial-gradient(at 80% 20%, rgba(6, 182, 212, 0.1) 0%, transparent 50%),
                 radial-gradient(at 50% 50%, rgba(168, 85, 247, 0.05) 0%, transparent 50%);
```

## Usage Guidelines

### Do
- Use teal for primary actions and brand moments
- Use coral sparingly for important callouts
- Maintain sufficient contrast ratios (4.5:1 minimum)
- Use muted variants for backgrounds, full colors for interactive elements

### Don't
- Use coral for primary actions (reserve for emphasis)
- Mix too many accent colors in one view
- Use light mode colors on dark backgrounds (or vice versa)
- Override semantic colors (error should always feel "error-like")

---

## Accessibility

All color combinations meet WCAG 2.1 AA standards:
- Primary text on background: 7:1+ contrast
- Secondary text on background: 4.5:1+ contrast
- Interactive elements: 3:1+ contrast against adjacent colors
